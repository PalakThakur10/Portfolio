# 💖 Palak Thakur Portfolio

A modern and responsive personal portfolio website built using **HTML and CSS**.
This project showcases my profile, skills, projects, and content creation work in a clean blush pink aesthetic design.

Hi, I'm **Palak Thakur**, a BCA 2nd semester student from **Chandigarh University**.

## ✨ Features

* 💻 Clean and modern UI design
* 🎀 Blush pink aesthetic theme
* 📱 Fully responsive (mobile-friendly)
* 🖼️ Profile image section
* 🌐 Social media links (Instagram, Threads, Facebook)
* 📚 Sections included:

  * About Me
  * Skills

  * Contact
